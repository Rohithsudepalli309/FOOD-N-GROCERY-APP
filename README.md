# F&G — Food & Groceries App

A full-featured React Native (Expo) mobile app for food delivery and quick grocery shopping.

## Getting Started

```bash
npm install
npm start
```

Then scan the QR code with **Expo Go** on your Android or iOS device.

## Project Structure

```
src/
  screens/       # All app screens
  components/    # Reusable components  
  navigation/    # Navigation setup
  store/         # Redux state
  services/      # API & Socket services
  hooks/         # Custom hooks
  constants/     # Theme, routes, API config
```
